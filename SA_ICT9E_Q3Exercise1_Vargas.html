HTML

HyperText Markup Language

It is used to create easy and basic web pages. It is a text file to apply a designed webpage, and easy to learn. Most browsers use HTML because, it is easy to learn and a free open source for everyone to use. Although it is mainly used for static web pages and older browsers can't be compatible with new features.

CSS

Cascading Style Sheets

It is a style sheet to layout webpages, it is usually used for text styles, background/text colors,table sizes and other designs for a webpage. CSS is a good file to design your webpages and add extra colors into your layouts.

Bootstrap

Bootstrap Version

Bootstrap is a free open source, for creating responsive websites and web applications. It is good for webpages to help fit the layouts in every devices like tablet, phone, pc, laptop and smart TV.
